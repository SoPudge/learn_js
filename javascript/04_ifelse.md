# Javascript基本语法-条件判断

## 语法及示例

```
> var bmi = 20
undefined
> 
> if (bmi <= 18.5) {
... console.log('过轻')
... } else if (bmi <= 25){
... console.log('正常')
... } else if (bmi <= 28){
... console.log('过重')
... } else if (bmi <= 32){
... console.log('肥胖')
... } else if (bmi <= 32){
... console.log('严重肥胖')
... }
正常

```

注意if else循环体当中，条件需要使用小括号，循环体需要使用大括号，永远不要省略语法。
